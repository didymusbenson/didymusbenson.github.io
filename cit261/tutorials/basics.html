Primatives

Lets start with the primative datatypes of JavaScript. You've got Boolean, Null, Undefined, Number, String, and Symbol which are your "primatives" (non-objects without methods).

Booleans hold a value of "true" or "false"; Nulls have one value: null; Undefined variables do not have a value at all.

Numbers have a specific definition that represent a value between -(253 - 1) and 253 - 1. There is no difference between integers and floats in JS.

Strings are made up of text data. There's a little more to it than that, but for our purposes, that's what you need to know.

You probably haven't heard of symbols yet. They are a new primative introduced in ECMAScript Edition 6. They are a unique and immutable value that can be used as the key of an Object. Symbols are called atoms. Here's the MDN page about symbols.

Objects (Arrays)

As MDN puts it: "an object is a value in memory which is possibly referenced by an identifier." For this review, we are most concerned with the following object types: Arrays, Dates, Functions, and JSONs (covered in another presentation).

An Array is a list of variables that can be accessed by an index. The array ["A", "B", "C"], for example contains three string values. The values of an array can be accessed using an index key. See the following code example:

var array = ["A", "B", "C"];
console.log(array[0]);

The output of this code would be "A". Arrays have several properties and methods that are helpful for manipulating the object data. The two properties that we care about are Array.length, which tells you how many items are in the array, and Array.prototype, which allows you to use certain methods on the individual values within the array.

Some Array.prototype methods to know are .push() (adds items to the end of the array), .pop() (removes the last element from the array and returns it), .indexOf() (tells you the first index of an element in the array with the value you send in), and .toString() (returns a string that represents all of the elements of the array).

You can learn more about arrays here on MDN.
