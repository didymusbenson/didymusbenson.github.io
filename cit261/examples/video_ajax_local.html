Video, Ajax, and Local Storage

In this example I've selected three public domain movies. You can do the following on this page:

  • "Learn more" about a movie. The page makes an API call to this tool and loads some information about the movie in question.
  • "Watch" the movie, which loads the movie up on the page for your viewing pleasure
  • "Favorite" the movie by clicking the star. It will save your preference to local storage (if you star a movie then leave and come back, it should remain starred)
